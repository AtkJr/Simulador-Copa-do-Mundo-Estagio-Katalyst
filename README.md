*⚽ Simulador de Copa do Mundo - Desafio Katalyst 2026*

Este projeto foi desenvolvido como parte do processo seletivo para a vaga de Estágio em Desenvolvimento de Software na **Katalyst**. O objetivo é simular uma Copa do Mundo completa, desde o sorteio de grupos até o grande campeão, integrando-se com uma API externa.

***O Projeto***

O simulador realiza o consumo de uma lista oficial de seleções, organiza-as aleatoriamente em grupos e processa todos os resultados de forma automática seguindo as regras oficiais da FIFA (Pontos, Saldo de Gols e Chaveamento de Mata-Mata).

***Referências: Copa de 2010***
Como um diferencial criativo, a interface e a paleta de cores foram inspiradas na **Copa do Mundo de 2010 na África do Sul**. 
* **Identidade Visual:** Uso de cores vibrantes (Amarelo, Verde e Vermelho) que remetem àquela edição.
* **Homenagem:** O projeto faz referências à icônica bola **Jabulani** e à energia das vuvuzelas que marcaram aquela época.

***Tecnologias Utilizadas***

**HTML5:** Estruturação semântica da página.
**CSS3:** Estilização personalizada com foco em responsividade.
**JavaScript (Vanilla):** Lógica de sorteio, simulação de partidas, cálculo de classificação e integração com API.
**API Externa:** Consumo e envio de dados via `fetch`.

***Funcionalidades implementadas***

1.  **Consumo de API (GET):** Busca dinâmica de todas as seleções participantes.
2.  **Sorteio Aleatório:** Distribuição justa das seleções em 8 grupos (A-H).
3.  **Simulação de Fase de Grupos:** Cálculo de vitórias, empates, derrotas e saldo de gols.
4.  **Mata-Mata Automatizado:** Chaveamento completo das oitavas de final até a grande final, com regra de desempate por pênaltis.
5.  **Integração de Resultados (POST):** Envio automático do campeão e detalhes da final para o servidor da Katalyst.
